## Setup and Boot Fixes

If Tails USB does not start, check firmware boot order, secure boot settings, and whether the USB drive was written with the recommended method from Tails documentation. Some machines need a boot menu key before Tails live USB appears.

If networking fails after boot, review wireless hardware support and try a wired adapter. Tails Tor requires a working network connection before privacy routing can begin, so connection setup matters.

If Tails persistent storage does not unlock, confirm the passphrase and make sure the USB drive is healthy. For any suspected corruption, consult Tails documentation before copying sensitive files.
